Pin Mode

Entering pin mode

You can use the mouse to pin a window by switching your desktop into pin mode. In this mode, the Windows cursor turns into a floating pin. Clicking on any window will place the pin on it and restore the cursor to its original shape. If the window you selected cannot be pinned for some reason, an error message will be displayed.

You can enter pin mode in several ways:

  • by clicking on the DeskPins icon in the system notification area (tray)
  • by right-clicking on the above icon and selecting "Enter pin mode" from the menu
  • via a hotkey; the default assignment is Ctrl-F11

To cancel pin mode without selecting a window, right-click anywhere with the mouse or hit any key on the keyboard.
