Tray Icon Menu

Tray menu commands

Right-clicking on the tray icon will bring up a menu with the following options:

Enter pin mode
Switches the desktop to Pin mode by changing the cursor into a floating pin, ready to be placed on a window. Selecting this menu item is the same as left-clicking on the tray icon.

Remove all pins
Removes all the pins currently used.

Options
Displays the Options dialog.

Help
Displays this help file.

About
Brings up a window that displays program information and the number of pins in use.

Close
Removes all pins and quits DeskPins.
