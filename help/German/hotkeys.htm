Hotkeys

Using hotkeys

If you prefer working with the keyboard, you can use the DeskPins hotkeys:

  • Ctrl-F11
    Enters Pin mode. This is especially useful on Windows XP where the DeskPins systray icon may be automatically hidden if not used for some time. Furthermore, you can use it to quickly pin several windows in succession without moving the cursor to the systray all the time to reenter pin mode.
  • Ctrl-F12
    Toggles the pin state of the currently active window. If there's already a pin on the window, it will be removed; otherwise a new one will be placed on it.

You can customize the hotkeys in the Hotkeys options page of the Options dialog.
